# Decision log

Settled decisions, ADR-lite: one dated entry each, with the rationale in one breath and a pointer to the document that owns the full treatment. When an [open question](./open-questions.md) is resolved, it graduates to an entry here — that is the only way a question leaves that file. A cold reviewer should be able to distinguish "considered and rejected" from "never considered" using this log alone.

Entries are append-only. Reversing a decision gets a *new* entry that supersedes the old one, never an edit.
